# stencil_lib
Generic parametric stencil generator for Python. 

Renders print-accurate transparent PNGs from real-world dimensions (cm/DPI) — ellipses (Trammel of Archimedes  ), Archimedean and golden spirals, spirographs, rose curves, and polygons. One engine, pluggable curve registry. 


## Integration with AR Apps
These stencils are designed for use with AR visualization tools. Overlay them on real INK's notebook paper surface https://sites.google.com/view/paperstackpro/home#h.s669t9tdrs5j 

Perfect for:
- Pre-visualization before cutting/engraving
- Pattern layout planning
- Digital-physical prototyping workflows
